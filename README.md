# 📱 App Mudança de Cor e Tela

Aplicativo Android desenvolvido em Kotlin que altera a cor de fundo da tela, pela cor que o usuário digitou.

## 🚀 Funcionalidades

- Entrada da cor
- Transferência dos dados para a segunda tela
- Exibição da cor digitada
